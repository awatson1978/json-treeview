json-treeview
========

JSON Treeview using Kendo UI.  You can find an online demo at http://json-treeview.meteor.com/  

Installation
------------------------


````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/json-treeview.git

# And then running it...
sudo mrt
````
